# Adam Cerha - Portfolio

A modern, responsive portfolio website showcasing my projects and skills.

## Features

- Clean, modern design
- Fully responsive layout
- Smooth scrolling navigation
- Animated project cards
- Easy to customize

## Setup

1. Clone this repository
2. Customize the content in `index.html` with your information
3. Update the contact links and project details
4. Push to GitHub

## GitHub Pages Deployment

This portfolio is configured to work with GitHub Pages:

1. Go to your repository settings
2. Navigate to "Pages" section
3. Under "Source", select the `main` branch
4. Click Save

Your portfolio will be live at: `https://joutau.github.io/portfolio/`

## Customization

- **Colors**: Edit CSS variables in `styles.css` (lines 7-13)
- **Content**: Update `index.html` with your information
- **Projects**: Add or modify project cards in the projects section
- **Contact**: Update contact links with your email, GitHub, and LinkedIn

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
